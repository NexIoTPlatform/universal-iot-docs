# 产品详情

## 功能说明

这是一个产品详情页面的示例。当前产品库使用模态框显示产品详情，而不是跳转到新页面。

## 产品信息

如果您需要创建独立的产品详情页面，可以：

1. **创建动态路由**：在 VitePress 中配置动态路由
2. **使用查询参数**：通过 URL 参数传递产品ID
3. **继续使用模态框**：当前的方式更简洁

## 当前实现

产品库现在使用模态框（弹窗）来显示产品详情，这样可以：
- 避免页面跳转
- 保持用户在当前页面的上下文
- 提供更好的用户体验

## 自定义

如果您希望使用独立页面，可以修改 `basic.vue` 中的 `navigateToProduct` 方法。
