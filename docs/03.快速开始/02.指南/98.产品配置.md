## 开启设备主动注册

- 平台默认只处理在数据库添加过的设备，如果你不想添加，请在`所有产品`->`产品详情`->`公共配置` 点击**编辑**->`自动注册新设备​`，选择 **是**


::: warning 注意
默认会给设备的`create_id`设置为`auto_register`,所以只有超级管理员有权限查看到
::::

- 当然你也可以手动添加设备，那么这个设备就会**划转**给你。

### 代码文件

```java
public class IoTDeviceAutoInsertIntercept implements IoTDownWrapper {

  @Resource private IoTDeviceMapper ioTDeviceMapper;
  @Resource private SupportMapAreasMapper supportMapAreasMapper;

  @Value("${iot.register.auto.unionId:auto_register}")
  private String autoRegister;

  @Value("${iot.register.auto.latitude}")
  private String latitude;

  @Value("${iot.register.auto.longitude}")
  private String longitude;

```

## 定位设备的实时位置更新


- 请在`所有产品`->`产品详情`->`公共配置` 点击**编辑**->`定位设备
`，选择 **是**

- 设备会根据实时上报的去当前的位置


::: warning 提示
物模型字段必须包含`coordinates`,且值必须是`longitude,latitude`，如`110.395041,22.062448`
::::
