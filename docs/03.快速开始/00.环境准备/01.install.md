## 📜 开源协议（AGPL3.0）说明

### Open Source ≠ Free Source

本项目采用开源方式发布，但**开源不等于免费**。请理解以下核心原则：

- **开源（Open Source）**：源代码公开，允许查看、学习、修改和分发
- **免费（Free Source）**：完全免费使用，无任何限制

本项目遵循开源精神，但保留完整的知识产权和专利保护。
