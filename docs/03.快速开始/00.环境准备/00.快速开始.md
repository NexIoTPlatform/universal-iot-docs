# 5分钟快速开始

## 🚀 一键部署

### Docker Compose 部署（推荐）

```bash
# 1. 下载部署脚本
curl -O https://nexiot.com/install.sh
chmod +x install.sh

# 2. 执行安装
./install.sh

# 3. 启动服务
docker-compose up -d

# 4. 访问系统
# 浏览器打开: http://localhost:8080
# 默认账号: admin / admin123
```

### 手动部署

```bash
# 1. 下载最新版本
wget https://nexiot.com/releases/nexiot-latest.jar

# 2. 创建配置文件
cat > application.yml << EOF
server:
  port: 8080

spring:
  datasource:
    url: jdbc:mysql://localhost:3306/nexiot
    username: nexiot
    password: nexiot123
  redis:
    host: localhost
    port: 6379

# 3. 启动服务
java -jar nexiot-latest.jar
```

## 📋 系统要求

- **Java**: JDK 21+
- **MySQL**: 5.7+ 或 8.0+
- **Redis**: 6.0+
- **内存**: 4GB+
- **磁盘**: 20GB+
