# 物模型数据调试

## 概述

本文档介绍MQTT设备接入中物模型数据的调试方法，包括数据格式验证、解析测试等内容。

#### 物模型主题模式，理论上不用解析协议，按照你的物模型上报数据即可

## 主要步骤

::: details 必看 {open}

- 1.准备MQTT工具
- 2.连接平台内置MQTT（配置文件）
- 3.开始上报数据
- 4.验证效果
  :::
